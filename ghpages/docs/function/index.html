    public init(userConfig: Object) source 

    import {init} from 'live2d-widget/src/index.js'

    The public entry point

    Params:

    NameTypeAttributeDescription
    userConfig Object
    • optional

    User's custom config
    model.jsonPath String
    • optional
    • default: ''

    Path to Live2D model's main json eg. https://test.com/miku.model.json
    model.scale Number
    • optional
    • default: 1

    Scale between the model and the canvas
    model.hHeadPos Number
    • optional
    • default: 0.5

    Horizontal position of model's head
    model.vHeadPos Number
    • optional
    • default: 0.618

    Vertical position of model's head
    model.myDefine Array
    • optional
    • default: []

    User's custom Defines which will override LDefine
    display.AA Number
    • optional
    • default: 2

    Antialiasing grade
    display.widget Number
    • optional
    • default: 150

    Widget to the canvas which shows the model
    display.height Number
    • optional
    • default: 300

    Height to the canvas which shows the model
    display.position String
    • optional
    • default: 'right'

    Left of right side to show
    display.hOffset Number
    • optional
    • default: 0

    Horizontal offset of the canvas
    display.vOffset Number
    • optional
    • default: -20

    Vertical offset of the canvas
    mobile.show Boolean
    • optional
    • default: true

    Whether to show on mobile device
    mobile.scale Number
    • optional
    • default: 0.5

    Scale on mobile device
    mobile.motion Boolean
    • optional
    • default: true

    Whether to enable motion detection on mobile devices
    name.canvas String
    • optional
    • default: 'live2dcanvas'

    ID name of the canvas
    name.div String
    • optional
    • default: 'live2d-widget'

    ID name of the div
    react.opacityDefault Number
    • optional
    • default: 0.7

    Default opacity
    react.opacityOnHover Number
    • optional
    • default: 0.2

    OnHover opacity
    react.myFunc Function
    • optional
    • default: func(e)

    Custom event handler, won't override main handler, will reveice the event type.
    dev.log Boolean
    • optional
    • default: false

    Whether to show log
    dev.mouseLog Boolean
    • optional
    • default: false

    Whether to show mouse log (tons of log), only work when dev.log is enabled
    dev.mouseFunc Function
    • optional
    • default: func(x, y, ix, iy)

    Custom logger, only work when dev.log is enabled, will receive (x, y, ix, iy), which presents the actucally position and vitural position
